# 多维度Tooltip功能使用指南
# Multi-Dimensional Tooltip Feature - Quick Guide

## 功能描述

在查看任何一个HSL分布对比图时，鼠标悬停在柱状图上会显示：
1. 照片缩略图
2. 其他两个维度的分布图

## 示例场景

### 场景1：查看 Hue Comparison（色调对比）

```
当前查看: Hue Comparison 图表
鼠标悬停在某个柱状图上
↓

┌─────────────────────────┐
│   照片缩略图 (200x200)   │
├─────────────────────────┤
│ Saturation Distribution  │
│ [迷你叠加柱状图]         │
├─────────────────────────┤
│ Lightness Distribution   │
│ [迷你叠加柱状图]         │
└─────────────────────────┘
```

**分析价值**: 
- 看到色调异常的照片
- 立即了解其饱和度和亮度分布
- 判断是否有过曝、欠曝或色彩饱和度问题

## 迷你图表内容说明

### Hue Distribution（色调分布）
- Very Red (0-10°, 350-360°) - 深红色
- Red-Orange (10-20°) - 橙红色
- Normal (20-30°) - 正常肤色
- Yellow (30-40°) - 偏黄
- Very Yellow (40-60°) - 非常黄
- Abnormal (60-350°) - 异常色调

### Saturation Distribution（饱和度分布）
- Very Low (<15%) - 非常低
- Low (15-30%) - 低
- Normal (30-50%) - 正常
- High (50-70%) - 高
- Very High (>70%) - 非常高

### Lightness Distribution（亮度分布）
- Low (<33%) - 暗部
- Mid (33-67%) - 中间调
- High (>67%) - 亮部

## 操作技巧

1. **快速扫描**: 在图表上移动鼠标，快速查看每张照片的多维度信息
2. **异常值定位**: 当发现某个维度异常时，通过tooltip立即了解其他维度是否也异常
3. **质量评估**: 综合三个维度判断照片质量
4. **对比分析**: 在相邻的柱状图之间移动，对比不同照片的特征

## 典型分析案例

### 案例1：肤色偏黄问题
```
在 Hue Comparison 中发现某照片 Yellow/Very Yellow 比例高
↓ 鼠标悬停
看到 Tooltip:
- Saturation: 正常分布
- Lightness: 中间调占主导
↓ 结论
这是典型的白平衡问题，不是曝光问题
```

### 案例2：过曝照片分析
```
在 Lightness Comparison 中发现某照片 High 比例异常高
↓ 鼠标悬停
看到 Tooltip:
- Hue: Abnormal 比例高
- Saturation: Very Low 比例高
↓ 结论
过曝导致高光区域失去色彩信息
```

### 案例3：理想照片识别
```
在任何 Comparison 图表中
↓ 鼠标悬停
看到 Tooltip 中所有三个维度都显示正常分布:
- Hue: Normal 占主导
- Saturation: Normal 到 High
- Lightness: Mid 占主导
↓ 结论
这是一张曝光和色彩都理想的照片
```

## 技术优势

✅ **零延迟**: 实时生成tooltip，无需等待
✅ **自适应**: 根据当前查看的图表智能显示另外两个维度
✅ **一致性**: 使用与主图相同的颜色和标签
✅ **专业性**: 符合现代数据分析工具的交互标准

## 快捷操作

| 操作 | 效果 |
|------|------|
| 鼠标悬停柱状图 | 显示tooltip |
| 鼠标移出图表区域 | 隐藏tooltip |
| 在柱状图之间移动 | 切换不同照片的tooltip |
| 切换标签页 | 查看不同主维度+另外两个辅助维度 |
