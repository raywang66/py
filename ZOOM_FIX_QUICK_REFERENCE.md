## 📏 动态圆角半径

```
100px缩略图 → 6px 圆角（成比例）
200px缩略图 → 10px 圆角（标准）
400px缩略图 → 12px 圆角（成比例）
```

**公式：** `radius = max(6, min(12, size * 0.05))`
